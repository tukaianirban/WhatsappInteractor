Android (Java) application to interact with Whatsapp application on a phone.

Launch the application, and then insert the fullname of a contact on the topmost text entry area (hint: Contact full name), and then press Resolve.
It resolves the contact's whatsapp details from the Android contacts on the phone. The contact's profile qualified domain name (as in WhatsApp) is then displayed in the textbox below.

After resolving the qualified domain name for a user:

To send a text message to the contact, type in a text in the textarea hinted with 'Text to send', and press on 'Send text'. This will open the Whatsapp conversation page for the contact, with the text you entered inserted into the input area there.
If you really want to send the text, then press then 'Send' button now on the whatsapp conversation page.

To make a voice call to the contact, press on 'VOIP CALL' button, and the Whatsapp call will launch automatically. (You should have resolve the contact before this point).

Same thing to be done to make a video call (press the 'VIDEO CALL' button).
